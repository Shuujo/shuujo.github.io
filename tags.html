Lists

Ordered Lists

Use the 'ol' and 'li' tags to create ordered lists within your webpage. For example:

<ol>
<li> </li>
<li> </li>
<li> </li>
</ol>

  1. Lorem ipsum dolor sit amet, consectetur adipiscing elit.
  2. Phasellus posuere ante quis ornare eleifend.
  3. Morbi sed erat nec dolor semper tincidunt eu id massa.
  4. Pellentesque porta felis aliquet molestie bibendum.
  5. Nullam quis turpis in dui condimentum bibendum dapibus vel quam.

Unordered Lists

Use the 'ul' and 'li' tags to create unordered lists within your webpage. For example:

<ul>
<li> </li>
<li> </li>
<li> </li>
</ul>

Bookmarks

Define bookmarks in your webpage with the 'id' attribute. Reference these 'id's using an 'a' tag and navigate to your bookmark by adding the 'href' attribute

<div id="[id_name]"> </div>

<a href="#[id_name]">

Tables

Using the 'table' tag, you can define your own table with your amount of headings, rows, and columns. The 'tr' tag defines a new row in the table. The 'th' tag defines a new header cell in the table. The 'td' tag defines a new cell in the table.

<table>
<tr>
<th>
</th>
</tr>
<tr>
<td>
</td>
</tr>
</table>
